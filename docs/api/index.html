Download OpenAPI specification:

License: Private hackathon prototype

OpenAPI contract for the Express routes mounted at /api in SprintPulse.

The API currently uses a personaId query or body field to resolve the acting user. Some routes are available only when ENABLE_MOCK_FLOW=true; in Supabase data mode those routes return 501 until their adapters are implemented.
